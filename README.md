# CSIFormer: A Transformer-Based Channel Estimation and Equalization Framework

## 项目简介
本项目旨在构建一个基于 Transformer 的联合信道估计与均衡模型，主要应用于 5G 高速铁路通信和车联网 V2X 场景。项目核心功能包括：
- 数据仿真与处理
- 模型构建与训练
- 实验结果分析与可视化
